# VIM 拾遺
 
* **使用%s/pattern/replace/g替換內容**
``` 
原文：
* 2019-12-06 08:20  MGFY

|   id  |  name  |  content |
| ----- | ------ | -------- |
| 001   | Abort  | Hi       |

* 2019-12-06 08:30  VIM

VIM命令：%s/\(\d\d\d\d-\d\d-\d\d\)/\*\*\1\*\*/g

执行结果：
* **2019-12-06** 08:20  MGFY

|   id  |  name  |  content |
| ----- | ------ | -------- |
| 001   | Abort  | Hi       |

* **2019-12-06** 08:30  VIM

其中 \(\d\d\d\d-\d\d-\d\d\)用于匹配日期，\1是匹配的结果。
如果有多个\(\) \(\) ...，最后匹配的结果为\1 \2 ...

```
